# Latihansoal1uklsedang
Latihan UKL 2
Program faktorial menggunakan fungsi faktorial(int n) untuk menghitung hasil perkalian semua bilangan dari 1 sampai n. 
Cara kerjanya adalah: 
-Pengguna memasukkan bilangan bulat positif. 
-Program memanggil fungsi faktorial(n). 
-Fungsi menghitung hasil perkalian dari 1 sampai n menggunakan perulangan for.
-Hasil perhitungan dikembalikan ke program utama. 
-Program menampilkan hasil faktorial ke layar.
